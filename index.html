Network Updates

Enter an API call to get information from the Ripple Network.
For example, try {"command": "ledger"} to get the date at which the last transaction was completed and how many transactions have been completed since you opened the connection.
Or {"command":"subscribe","id":0,"streams":["ledger"]} to get the most recent ledger hash.
Otherwise, here is a list of other ripple-lib API calls that can be called on the Ripple Network: ripple-lib APIs. The rest of the API calls will return the default objects that they are supposed to return.
